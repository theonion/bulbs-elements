# `<bulbs-pinned-element>`

Attributes
  * `pinned-to` *(required)* selector of the element pinned to. This should be a parent of `<bulbs-pinned-element>`


# Example
```html
<parent-element class="foobar">
  <bulbs-pinned-element pinned-to=".foobar">
  </bulbs-pinned-element>
</parent-element>
```
